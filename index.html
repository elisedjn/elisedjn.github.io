Web

Aide au design de flakyci.com

Flakyci est un outil destiné aux développeurs web qui souhaitent optimiser leur code en réduisant le nombre de tests qui ont un résultat aléatoire (flaky tests).
Ce projet est toujours en cours de réalsation.
J'aide au design des différentes parties du site web (page vitrine et dashboard) avec le framework Tailwindui. J'utilise également Visual Studio Code et Github pour le suivi du projet.
